To check if mongodb is installed.

> mongo

MongoDB shell version v3.6.3
connecting to: mongodb://127.0.0.1:27017
MongoDB server version: 3.6.3
Server has startup warnings: 
...

To show existing databases,

> show dbs

admin   0.000GB
config  0.000GB
local   0.000GB

To create a database/ use a database,

> use mystudents

switched to db mystudents

> show dbs
admin   0.000GB
config  0.000GB
local   0.000GB

NOTE: You cannot see the created database as it is empty.
